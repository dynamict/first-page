Class Presentations: Spotify

Let's take a deep dive into the process of creating these slides!

To follow along, you can download the presentation here

This presentation was used to complete a case study in class related to Spotify and how their situation in 2016/2017 could lead to further growth or turmoil in the future. Although I could have taken a more traditional presentation making process to create the slides, I wanted to stretch my creative freedoms in a class project to show my abilities on Microsoft PowerPoint.


To allow myself to create the slides that I envisioned, I first had to create the canvas on which I would be able to operate on. Taking inspiration from Spotify’s PC interface, I created a slide template that allowed me to creatively imitate the look and feel of Spotify’s style. This was done by utilizing PowerPoint’s ‘Master Slide’ feature which allowed me to quickly create a consistent template.


Adding a little personal flavour to the presentation slides, I wanted to create something that would lighten the mood when presenting. Therefore, I created album covers in the art style that Spotify tends to use. Although this seems like a simple task, all artistic and style features that are implemented within the slides are made from PowerPoint, no use of Photoshop was present throughout this entire presentation!


Utilizing new features that I never used before, I was able to create different ‘hover actions’ throughout the presentation. This meant that during the presentation, I would be able to hover over certain aspects of the slide deck and create animations that could mimic how the Spotify PC app would operate. For example, as a table of contents in a “browse” menu, you would be able to hover over different album covers and the text inside would underline.
